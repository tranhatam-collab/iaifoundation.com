Initial foundation release for IAIFoundation.com and the early Intent OS monorepo.

- expanded public website with docs, partners, SEO, accessibility, and governance content

- shared contracts, DB layer, runtime-core, policy engine, observability, and audit primitives
- SQL migrations, OpenAPI specs, Vitest setup, and baseline tests
- multi-layer AI governance documentation across Level 1, Level 2, and Level 3
